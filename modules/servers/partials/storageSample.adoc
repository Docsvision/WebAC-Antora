== Пример. Хранилище файлов в {location}

Далее приведён пример настройки хранилища файлов {dv} в {location}.

include::partial$excerpts.adoc[tags=openStorageSettings]
+
. xref:storage-setup.adoc[Подключите] к {dv} хранилище в {location}.
+
.Параметры подключения настройте следующим образом:
.. В поле _Название_ введите `{s-name}` или любое другое название.
.. В списке _Тип_ выберите значение *_{type}_*.
.. В списке _Состояние_ выберите значение *_Online_*.
.. В поле _Макс. размер_ оставьте значение по умолчанию (`0`).
+

Когда объём хранилища закончится, попытка добавления файла в него завершится ошибкой.
+
.Чтобы избежать ошибок, ограничьте размер хранилища:
** В поле _Макс. размер_ укажите примерный объем свободного места, который доступен для FileStream.
** В списке _Состояние_ выберите значение *_Auto_*.
+
.. Флаги в блоке _Разделы_ установите по собственному усмотрению. Флаг `*Временный*` рекомендуется снять.

. xref:storage-setup-group.adoc[Создайте] группу хранилищ.
+
.Параметры группы настройте следующим образом:
* В поле _Название_ введите `{sg-name}` или любое другое название.
* В списке _Режим выбора_ оставьте значение по умолчанию.
+
.Пример настройки группы хранилищ
image::dev@platform:admin:{sg-img}.png[Пример настройки группы хранилищ]
+
NOTE: Хранилище можно включить в существующую группу, тогда создавать новую не потребуется.
+
. xref:storage-add-to-group.adoc[Включите] хранилище `{s-name}` в группу `{sg-name}` и xref:storage-rules.adoc[настройте] правила помещения файлов в группу хранилищ.
+
--
.Параметры правила настройте следующим образом:
* В поле _Название_ введите `{r-name}` или любое другое название.
* В списке _Группа_ выберите значение *_{sg-name}_*.
* В списке _Тип_ оставьте значение *_Все файлы_* или конкретизируйте параметры файлов (см. рисунок ниже), которые будут размещаться в хранилище.
--
