= Назначить базу данных по умолчанию

include::partial$excerpts.adoc[tags=maindb]

.Чтобы назначить имеющуюся базу данных основной:
. Откройте страницу menu:Серверы[Имя-сервера > Сервер приложений > Базы данных].
+
.Раздел "Базы данных" консоли настройки {dv}
image::dev@platform:admin:settings-database.png[Раздел "Базы данных" консоли настройки {dv}]
+
. Для нужной базы данных в столбце _По умолчанию_ нажмите кнопку *Назначить*.
. Появится окно с вопросом `Вы действительно хотите назначить DvTest в качестве базы по умолчанию?` и двумя вариантами *Назначить* и *Отмена*.
. Нажмите *Назначить*, чтобы сохранить настройки. База данных будет назначена основной, веб-сервер {dv} будет работать с ней по умолчанию. Перезапустите *{sss-new}* на Linux.
+
При выборе *Отмена* изменения будут отменены.
