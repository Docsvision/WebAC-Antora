= Включить хранилище в группу хранилищ

Администратор должен включить хранилище в группу хранилищ, иначе оно не будет доступно для сохранения бинарных данных файлов.

[#add]
== Добавление хранилища в группу

.Чтобы включить хранилище в группу хранилищ:
include::partial$excerpts.adoc[tags=openStorageSettings]
+
.Окно выбора хранилища для добавления в группу хранилищ
image::dev@platform:admin:select-storage.png[Окно выбора хранилища для добавления в группу хранилищ]
+
. Выберите добавляемое хранилище в списке _Хранилища_.
. Перетащите строчку хранилища в список _Группы хранилищ_.
+
Группа хранилищ может содержать одно или несколько хранилищ.
+

[#exclude]
== Исключение хранилища из группы

.Чтобы исключить хранилище из группы:
. Найдите хранилище в таблице _Группы хранилищ_ и нажмите на кнопку image:buttons/x.png[Крест] в правой части строки.
+
При удалении хранилища из группы появляется окно с сообщением: `Вы уверены, что хотите исключить хранилище из группы?` и кнопками *Да* / *Нет*. После подтверждения хранилище удаляется из группы, но не из списка хранилищ.
+
NOTE: После исключения хранилища из всех групп, оно перестанет использоваться для сохранения новых бинарных данных, но доступ к имеющимся в хранилище данным сохранится.
