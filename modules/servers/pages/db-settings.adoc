Индикатор состояния БД::
В зависимости от текущего состояния БД индикатор может менять цвет:
+
* Серый -- БД в состоянии создания, обновления, подключения,
* Зеленый -- БД уже подключена и доступна для соединения,
* Красный -- БД уже подключена, но по какой-то причине недоступна для соединения,
* Пустой -- БД подключена, но с сервера не вернулась информация.
