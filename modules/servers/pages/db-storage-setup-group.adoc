[#edit]
== Изменение настроек группы хранилищ

.Чтобы изменить настройки группы:
. Выберите группу в списке _Группы хранилищ_, нажмите на её название.
. Измените и сохраните настройки.
+
Тип хранилища не может быть изменён.

[#delete]
== Удаление группы хранилищ

.Чтобы удалить группу хранилищ:
. Найдите группу хранилищ в списке и нажмите на кнопку image:buttons/x.png[Крест] в правой части строки.
+
При удалении группы появляется окно с сообщением: `Вы уверены, что хотите удалить группу хранилищ?` и кнопками *Да* / *Нет*. После подтверждения группа удаляется из списка. Хранилища, добавленные в группу не затрагиваются.
+
[WARNING]
====
Перед удалением группы хранилищ убедитесь, что она не используются по умолчанию для хранения типов файлов.
====
